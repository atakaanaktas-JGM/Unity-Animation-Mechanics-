# Unity Animation & Timeline Experiments

This repository contains animation and gameplay mechanic experiments developed with Unity.

The focus of this project is to explore Unity's animation systems such as Animator Controllers and Timeline sequences.

## Implemented Systems

- Timeline based animation control
- Animator state machines
- Enemy animation sequences
- Player animation triggers
- Camera positioning and cinematic control
- Event-driven animation timing

## Technologies

- Unity
- C#
- Unity Timeline
- Animator Controller

## Purpose

This project serves as a practice environment for implementing and testing gameplay animation systems and cinematic timing using Unity tools.
